# Adote Pet — Quality Bar (Obrigatório)

## Definition of Done por entrega
- Responsivo: desktop e mobile.
- Estados: loading/empty/error/success implementados.
- Tokens respeitados (grid 8, radius 8/16, border 1).
- Acessibilidade mínima (labels/aria/alt text).
- Performance: evitar imagens pesadas; lazy-load quando aplicável.

## Evidências obrigatórias
- Produzir 1 walkthrough de como testar.
- Produzir screenshot “antes/depois” quando a mudança for UI.
- Se houver mudanças grandes, gerar um plano + task list antes de codar.
